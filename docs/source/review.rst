การประเมินความพึงพอใจ
===================

* แบบประเมินจะปรากฏต่อเมื่อทำกิจกรรมเหล่านี้ครบ 3 ครั้ง

    * Generate template
    * Validate template
    * Save to cloud

* ให้คะแนนได้ตั้งแต่ 0 - 5 คะแนน
* หากให้คะแนนช่วง 0 - 4 คะแนน จะต้องกรอกข้อเสนอแนะด้วย
