การเพิ่มชื่อคณะกรรมการสอบวิทยานิพนธ์ผ่าน iThesis Web Add-in¶

ขั้นตอนการเพิ่มชื่อคณะกรรมการสอบ¶

  1. กด add committee

  2. เลือกประเภทของกรรมการ

  3. เลือกชื่อของคณะกรรมการ (สามารถเลือกได้เฉพาะชื่อที่มีในระบบเท่านั้น)

  4. เพิ่มตำแหน่งและวุฒิกรรมการสอบ

  5. กด save เพื่อบันทึกรายชื่อกรรมการ

การเพิ่มตำแหน่งและวุฒิกรรมการสอบ¶

  1. กด add data ตำแหน่งที่ต้องการ

  2. เลือกตำแหน่งหรือวุฒิการศึกษา

  3. กด add
